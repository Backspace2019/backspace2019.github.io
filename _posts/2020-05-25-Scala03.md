运算符的本质

在Scala中其实是没有运算符的，所有运算符都是方法。

- scala是完全面向对象的语言，所以数字其实也是对象
- 当调用对象的方法时，点.可以省略
- 如果函数参数只有一个，或者没有参数，()可以省略

```scala
object ScalaOper {
  def main(args: Array[String]): Unit = {
​    val i : Int = 10
​    val j : Int = i.+(10)
​    val k : Int = j +(20)
​    val m : Int = k + 30
​    println(m)
  }
}
```

------

运算符的使用和Java运算符的使用基本相同，只有个别细节上不同。

++运算符

scala中没有++运算符，因为有歧义。

```java
int i=1;
i=i++;
//先赋值，再运算，所以i=1
```
