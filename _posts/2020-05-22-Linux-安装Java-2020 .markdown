### 解压JDK安装包

1.使用xftp连接Linux，将JDK的安装包上传到`/opt/software`文件夹下

2.使用xshell连接Linux[非root用户]，`cd /opt/sofware`来到software目录

3.解压安装包到/opt/module目录下 `tar -zxvf jdk1.8.0_144.tar.gz -C /opt/module`
